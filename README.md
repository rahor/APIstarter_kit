# Node JS API starter code

To start this project, clone the repo and follow those steps:
```
cd path/to/your/directory
git init
npm i
touch .env
```

Add those environement variables:
```
# Mongo
MONGO_URL = <YOUR-MONGO-URL>

# Server
PORT= <YOUR-IP>


# JWT
JWT_SECRET = <YOUR-JWT-SECRET>
```
npm start
